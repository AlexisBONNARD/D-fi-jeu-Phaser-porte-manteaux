Petit jeu développé avec **Phaser.js** dans le cadre d’un défi de développement.  
Le projet consiste à créer un jeu 2D interactif autour de l'objets (porte-manteaux) et le tout en 2h.

---

# Description

Ce projet permet de découvrir les bases du **développement de jeux en JavaScript** avec Phaser :

- gestion des **scènes**
- utilisation de **sprites**
- gestion du **clavier**
- **collisions et interactions**
- organisation d’un projet de jeu web

Le jeu fonctionne directement dans un **navigateur web**.

---

# Technologies utilisées

- **JavaScript**
- **HTML5**
- **Phaser.js**

# Installation et lancement
## Cloner le projet
```bash
git clone https://github.com/AlexisBONNARD/D-fi-jeu-Phaser-porte-manteaux.git
```
Entrer dans le dossier
```bash
cd D-fi-jeu-Phaser-porte-manteaux
```


Installe Visual Studio Code
Installe l’extension Live Server

Clique droit sur Demo.html
Clique sur "Open with Live Server"

Le jeu s’ouvre automatiquement dans le navigateur :
http://127.0.0.1:5500
